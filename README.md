# 🚀 Projeto Conversor de Moedas

Uma aplicação web simples e funcional que permite ao usuário converter valores entre diferentes moedas de forma interativa e visual.  
Desenvolvido com foco em **Front-End** utilizando **HTML, CSS e JavaScript**.

---

## 📌 Sobre

Este projeto foi criado como exercício prático de desenvolvimento web para demonstrar habilidades no uso de **HTML5, CSS3 e JavaScript**.  
O conversor calcula o valor entre moedas baseadas nas seleções feitas pelo usuário.

---

## 🧠 Funcionalidades

✨ Permite converter valores entre diferentes moedas  
🎨 Interface amigável e responsiva  
📱 Compatível com navegadores modernos  
🚀 Projeto leve e de rápida execução
